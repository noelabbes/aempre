\section{Propósito y Enlace de Gobernanza}
El presente documento constituye el Índice Maestro y el Punto de Acceso Oficial al Repositorio de Arquitectura del Proyecto SIPBA de OSIPTEL. Siguiendo los principios de agilidad y gobernanza tecnológica viva (\textit{Architecture-as-Code}), todos los activos arquitectónicos, diagramas de metamodelos, estándares tecnológicos y registros de decisiones se encuentran bajo control de versiones y almacenamiento seguro en el repositorio central de GitHub. Este documento sirve como puente metodológico alineado a TOGAF para auditar y navegar dichos componentes.
